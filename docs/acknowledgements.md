# Acknowledgements

## Purpose

`ui-motion-workflow` is an orchestration layer, not a reinvention of the upstream open-source ecosystem it builds on.

This project benefits from the work of maintainers and contributors behind the reference-stack tools it can coordinate.

## Upstream Thanks

Special thanks to the maintainers and contributors of projects that inspire or complement this workflow, including:

- `ui-ux-pro-max`
- `react-bits`
- `magicui`

These projects make it easier to build richer UI systems, stronger motion direction, and better frontend implementation workflows.
