# React Bits Provider

## Role

`react-bits` is the expressive provider in the reference stack.

Use it when the UI needs a visible motion signature rather than only structural polish.

## Best Fit

- hero sections
- showcase blocks
- ambient backgrounds
- expressive promotional components
- memorable animated text or interaction moments

## Strengths

- strong visual identity
- broad catalog of animated surfaces
- good fit for marketing-heavy pages

## Risks

- can dominate a dense product interface
- easier to over-stack with other motion layers

## Prefer It When

- the page can support one strong motion focal point
- the user wants a more memorable or cinematic effect
- the design direction explicitly allows spectacle

## Avoid It When

- the surface is already crowded
- the UI needs only subtle polish
- the right answer is embedded motion within an existing component
