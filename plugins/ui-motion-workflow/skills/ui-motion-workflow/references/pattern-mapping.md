# Pattern Mapping

## Premium Hero

Direction:

- restrained
- editorial
- confident

Likely chain:

1. `ui-ux-pro-max` for tone and hierarchy
2. provider choice:
   - `react-bits` for `BlurText`, `SplitText`, `SoftAurora`, or `GlassSurface`
   - `magicui` for a more product-polished hero block if spectacle should stay controlled
   - `motion-primitives` if the hero structure already exists and only the motion layer needs refinement
3. browser validation to ensure the hero stays readable

## Playful CTA Section

Direction:

- energetic
- vivid
- interactive

Likely chain:

1. `ui-ux-pro-max` for motion intensity limits
2. provider choice:
   - `react-bits` for `Magnet`, `BubbleMenu`, `ColorBends`, or `GradientText`
   - `magicui` if the CTA section should stay more product-like
   - `motion-primitives` for hover or reveal polish on an existing CTA component
3. browser validation to make sure it does not overwhelm the CTA

## Technical / Futuristic Surface

Direction:

- precise
- high-tech
- immersive but legible

Likely chain:

1. `ui-ux-pro-max` for technical styling constraints
2. provider choice:
   - `react-bits` for `DecryptedText`, `DotField`, `Ferrofluid`, or `LaserFlow`
   - `motion-primitives` for restrained technical motion inside a product UI
3. browser validation to check readability and performance feel

## Subtle Product Polish

Direction:

- quiet
- trustworthy
- low-distraction

Likely chain:

1. `ui-ux-pro-max` if restraint rules are needed
2. provider choice:
   - `react-bits` for `FadeContent`, `CountUp`, `PillNav`, or `DarkVeil`
   - `magicui` as the preferred default if the surface is product-facing and needs stronger structural polish
   - `motion-primitives` if only micro-motion is missing from an otherwise correct local component
3. browser validation to confirm the effect is present but not dominant
